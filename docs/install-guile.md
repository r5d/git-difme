# install guile

On OpenBSD:

    # pkg_add -v guile

On GNU Guix:

    $ guix package -i guile

On Debian GNU/Linux qbased systems:

    # apt-get install guile

on arch based systems:

    # pacman -S guile

on macOS, install [homebrew][], then:

    $ brew install guile

[homebrew]: http://brew.sh
